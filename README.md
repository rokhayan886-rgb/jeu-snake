Snake – Portfolio Version
Description

Ce projet est une version interactive du jeu Snake en Python avec interface graphique Tkinter.
Le joueur contrôle un serpent qui doit manger des pommes pour grandir tout en évitant de se heurter aux murs ou à lui-même.

Le jeu est conçu pour être simple à lancer et à jouer, parfait pour ton portfolio.

Fonctionnalités

Déplacer le serpent avec les flèches du clavier.

Le serpent grandit en mangeant les pommes.

La partie se termine si le serpent touche les murs ou lui-même.

Reset du jeu possible pour recommencer.

Technologies utilisées

Python 3

Tkinter pour l’interface graphique

Concepts utilisés : classes, événements clavier, gestion des collisions

Installation et utilisation

Télécharger le dépôt sur ton ordinateur :

git clone https://github.com/rokhayan886-rgb/jeu-snakes.git

Aller dans le dossier du jeu :

cd jeu-snakes/games/snake

Lancer le jeu (assurez-vous que Python 3 est installé) :

python SnakeGame.py

Utilisez les flèches pour déplacer le serpent.
Mangez les pommes pour grandir et évitez les collisions !
